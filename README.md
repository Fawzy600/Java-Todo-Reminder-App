# 📋 Java Todo List with Reminders
A desktop application built with Java Swing that helps users manage tasks and alerts them when deadlines pass.

### ✨ Features
* **Task Management:** Add, Edit, and Delete tasks.
* **Smart Reminders:** Background service checks for overdue tasks every minute.
* **Visual Cues:** Overdue tasks are highlighted in red automatically.
* **Status Tracking:** Change tasks to Pending, In-Progress, or Done.

### 🛠️ Built With
* Java 17+
* Maven
* Swing (GUI)
